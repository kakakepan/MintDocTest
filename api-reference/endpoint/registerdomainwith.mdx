title: "Register a Domain with Contact Creation"
openapi: "POST /api/v2/domains/{domain}/register_and_create_contacts"
sidebarTitle: "Register a Domain with Contact Creation"
description: This endpoint facilitates domain registration and contact creation by performing the following steps:

        1. Checks domain availability at the registry.
        2. Creates new contacts for the reseller (if needed).
        3. Creates contact handles in the registry.
        4. Starts domain registration at the registry.
