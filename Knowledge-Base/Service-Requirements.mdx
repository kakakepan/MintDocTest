The **"Service Requirements"** option within the **"Knowledge Base"** refers to the specific conditions, criteria, or prerequisites that must be met to register a domain name successfully. It is important to carefully review and meet the service requirements before initiating the domain registration process to ensure a smooth and successful registration.

# Additional Fields

Within the **"Service Requirements"** option, the **"Additional fields"** refer to the specific information required beyond the basic command parameters to process the command successfully. They are necessary for certain domain extensions and registration scenarios based on the owner's country and entity type.

The additional fields required vary depending on the command and extension, taking into account the owner's country and entity type. For more detailed information about the specific additional fields required for each command or extension, please refer to the **"Settings_AdditionalFields"** sheet. This sheet provides explanations for each field, including **"profile_type_id"** indicating the entity type (company or individual), **"additional_field_type"** specifying the data type, and **"countries"** specifying the applicable countries. The table below illustrates the additional fields required for domain registration based on the domain extension, owner's country, and entity type:

| Domain Extension | Owner's Country | Entity Type | Additional Fields Required                              |
|------------------|-----------------|-------------|---------------------------------------------------------|
| .FI              | Finland         | Individual  | Date of Birth (yyyy/mm/dd)   HETU/Finnish identity code |
| .FI              | Any             | Individual  | Date of Birth (yyyy/mm/dd)                              |
| .com.br          | Brazil          | Company     | CNPJ number                                             |

Users can ensure compliance with registration requirements by providing the necessary additional fields and successfully completing the domain registration process for specific extensions and registration scenarios.

# Additional Field Types

Within the **"Service Requirements"** option, the **"Additional fields Types"** refer to the **"additional_field_type"** in the **"Settings_AdditionalFields"** sheet, which categorizes the type of field required for additional information during domain registration. It helps identify the specific nature of the additional field based on various extensions and registration scenarios.

Company Number is an additional field type attributing to a unique identification number assigned to a company or organization. To provide a more precise understanding, here is a table illustrating additional field type **"Company Number"** and its corresponding types with their descriptions:

| Company Number Types | Description                                                                                                                                                         |

# Additional Documentation

Within the **"Service Requirements"** option, the **"Additional documentation"** refers to the supplementary paperwork or files that may be required for certain services and extensions. The list of additional documentation for each service can be found in the **"Settings_AdditionalDocumentation"** sheet. The following explanations are provided for better understanding:

| Additional Documentation         | Description                                                                                                                                                                    |
| :------------------------------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| accept_scanned_copy              | Indicates whether scanned copies of documentation are accepted. If marked as "yes," scanned copies can be submitted. This may affect the format or structure of request forms. |
| additional_documentation_file_en | Provides a downloadable template document in English that serves as a reference or guide for the required documentation.
