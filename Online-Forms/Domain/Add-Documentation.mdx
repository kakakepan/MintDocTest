# Add Documentation

To add documentation to a domain, follow the steps below:

1. Navigate to the homepage and locate the **"Online Forms"** option.
2. Click on the **"Online Forms button"** to proceed.

![29](/images/domainimages/29.png)

3. Select the **Domains** button by clicking on it.

![30](/images/domainimages/30.png)

3. Select **Add Documentation**.

![31](/images/domainimages/31.png)
