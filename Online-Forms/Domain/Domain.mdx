# Overview

The domain feature offered by Online Forms enables resellers to conveniently oversee multiple domains through a unified platform. This feature is an efficient tool for managing domains and empowers resellers to deliver website hosting services for their respective domains. Within the domain feature, users can perform the following actions:

| Feature            | Description                                                                                                                             |
| :----------------- | :-------------------------------------------------------------------------------------------------------------------------------------- |
| Search             | Search any domain that has been registered within the reseller dashboard.                                                               |
| Register           | Register new domain to the reseller dashboard.                                                                                          |
| Modify             | Modify the registered domain.                                                                                                           |
| Renew              | Renew any registered domain within the reseller dashboard.                                                                              |
| Delete             | Delete any registered domain along with its configuration settings.                                                                     |
| Add Information    | Add any additional domain-related information. Using this feature, you can also view the status of requests to submit information.      |
| Add Documentation  | Add documentation to any domain that is registered. Using this feature, you can also view the status of a request to add documentation. |
| Create Domain Host | Create a host for the registered domain.                                                                                                |
| Modify Domain Host | Modify a registered domain's host allowing the flexibility to remove or modify hosts.                                                   |
