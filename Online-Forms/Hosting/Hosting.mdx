# Overview

Hosting is a feature of the reseller dashboard that allows the supersellers to host the resellers' domain. This feature allows the resellers to manage their domain and its hosting in a single place. By using this feature, the resellers can do the following:

| Feature | Description                                                 |
| :------ | :---------------------------------------------------------- |
| Add     | Add the resellers' domain to be hosted by the supersellers. |
| Modify  | Modify the hosting configurations.                          |
| Renew   | Renew the hosting duration.                                 |
