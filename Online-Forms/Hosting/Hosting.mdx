**"Hosting"** is a feature of the reseller dashboard that allows the supersellers to host the resellers' domain. This feature allows the resellers to manage their domain and its hosting in a single place. By using this feature, the resellers can do the following:

<CardGroup cols={2}>
  <Card
    title="Add"
    icon="square-plus"
    href="/Online-Forms/Hosting/Add"
  >
    Add the resellers' domain to be hosted by the supersellers.
  </Card>
  <Card title="Modify " icon="wrench" href="/Online-Forms/Hosting/Modify">
    Modify the hosting configurations. 
  </Card>
  <Card title="Renew " icon="layer-plus" href="/Online-Forms/Hosting/Renew">
    Renew the hosting duration.
  </Card>
</CardGroup>
