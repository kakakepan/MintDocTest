# Overview
A contact is a group of data related to a person or company (organization) related to the domain names. The resellers can add or delete the contact related to their domain in this contact feature.
Within the contact feature, users can perform the following actions:

| Feature            | Description                                                                                                                             |
| :----------------- | :-------------------------------------------------------------------------------------------------------------------------------------- |
| Create            | Create contacts related to the domain.                                                              |
| Modify          | Modify the contacts.                                                                                          |
