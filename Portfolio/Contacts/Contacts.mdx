# Overview

The **"Contacts"** feature within the **"Portfolio"** facilitates the management of contacts. Contact is a specific role or category associated with the domain or hosting account. Resellers can manage and check their contacts through a centralized platform.

# Manage Contacts

To perform a check and **"Manage contacts"** associated with a particular domain, please follow the steps outlined below:

1. Click on the "**Contacts"** option under the **"Hosting"** option , which leads to the following:

![1](/images/portocontactsimages/1.png)

2. A popup will appear when you select the  **"Search Filters"** option.

![2](/images/portocontactsimages/2.png)

3. The **"Search Filters"** allow you to specify various parameters such as the following:

| Parameter           | Description                                                                                                                        |
| :------------------ | :--------------------------------------------------------------------------------------------------------------------------------- |
| Handle ID           | A unique identifier assigned to a contact or entity within a domain or hosting system for reference and identification purposes.   |
| Organization Name   | The name of the organization or company associated with the contact.                                                               |
| Email               | The email address linked to the contact.                                                                                           |
| Country             | The country or geographic location associated with the contact's address.                                                          |
| Last Name           | The surname or family name of the individual contact.                                                                              |
| First Name          | The first name of the individual contact.                                                                                          |
| Internal Contact ID | An internal identification code or number assigned to the contact within the domain or hosting system for administrative purposes. |

![3](/images/portocontactsimages/3.png)
