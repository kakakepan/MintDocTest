To identify the location, server, and status of the domain's website or email services hosting, follow the steps stated underneath:

1. Click on the "**Hosts"** option under the domain portfolio, which provides information such as the following: 

| Parameter       | Description                 |
| :-------------- | :-------------------------- |
| Reseller E-Mail | The email of the reseller.  |
| Host Name       | The name of the host.       |
| Host IP         | The IP address of the host. |
| Status          | The status of the host.     |

![7](/images/portodomainsimages/7.png)

2. A popup will appear when you select the  **"Search Filters"** option.

![8](/images/portodomainsimages/8.png)

3. The **"Search Filters"** allow you to specify various parameters including the following: 

| Parameter | Description                                                                                                           |
| :-------- | :-------------------------------------------------------------------------------------------------------------------- |
| Host Name | The name of the host.                                                                                                 |
| DNS IP    | The IP address of the host.                                                                                           |
| Status    | The status of the host. It can be specified with "In Process", "Active", "Disabled", "Cancelled by RSS", and "Reversed by RSS". |

![9](/images/portodomainsimages/9.png)
