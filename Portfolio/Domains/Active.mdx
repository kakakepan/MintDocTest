To check the **"Active"** status of your domain of the website or online service, follow the steps stated underneath:

1. Click on the "**Active"** option under the domain portfolio, which provides information such as the following:

| Parameter       | Description                                         |
| :-------------- | :-------------------------------------------------- |
| Domain Name     | The name of the domain.                             |
| Status          | The status of the domain.                           |
| Expiration Date | The expiration date of the domain.                  |
| Transfer Lock   | The domain configuration of the transfer lock       |
| Auto Renew      | The domain's configuration to automatic renewal. |

![1](/images/portodomainsimages/1.png)


2. A popup will appear when you select the  **"Search Filters"** option.

![2](/images/portodomainsimages/2.png)


3. The **"Search Filters"** allow you to specify various parameters such as the following:

| Parameter       | Description                                                                             |
| :-------------- | :-------------------------------------------------------------------------------------- |
| Domain Name     | The name of the domain.                                                                 |
| Status          | The status of the domain. It can be specified with "Registered" or "Soon to be Expired".     |
| Expiration Date | The expiration date of the domain.                                                      |
| Transfer Lock   | The domain configuration of the transfer lock. It can be specified with "Yes" or "No".      |
| Auto Renew      | The domain's configuration to automatically renews. It can be specified with "Yes" or "No". |

![3](/images/portodomainsimages/3.png)
