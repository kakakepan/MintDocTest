To verify the **"Pending"** status of the domain's registration or management status, follow the steps stated underneath:

1. Click on the **"Pending"** option under the **"Domains Portfolio"**, which provides information such as the following:

| Parameter    | Description                        |
| :----------- | :--------------------------------- |
| Date         | The pending date of the domain.    |
| Domain Name  | The name of the domain.            |
| Command      | The pending command of the domain. |
| Missing Data | The missing data from the domain.  |

![10](/images/portodomainsimages/10.png)

2. A popup will appear when you select the  **"Search Filters"** option.

![11](/images/portodomainsimages/11.png)

3. The **"Search Filters"** allow you to specify various parameters including the following:

| Parameter   | Description                                                                                                                                             |
| :---------- | :------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Domain Name | The name of the domain.                                                                                                                                 |
| Command     | The pending command of the domain. It can be specified with **"Register Domain"**, **"Transfer In Domain"**, **"Add Hosting"**, **"Add Productivity Suite"**, and **"Add SSL Cert"**. |

![12](/images/portodomainsimages/12.png)
