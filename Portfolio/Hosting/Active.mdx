To check the **"Active"** domain hosting status of the website or online service, follow the steps stated underneath:

1. Click on the "**Active"** option under the hosting portfolio, which leads to the following:

| Parameter         | Description                                                                                                               |
| :---------------- | :------------------------------------------------------------------------------------------------------------------------ |
| Domain            | The name and address that uniquely identifies a website on the internet.                                                  |
| Status            | The current state or condition of the domain.                                                                             |
| Hosting Type      | The method or technology used to store and serve website files.                                                           |
| Hosting Brand     | The company or provider that offers the hosting services for the website.                                                 |
| Hosting Plan      | The specific package or tier of hosting services.                                                                         |
| Accounts          | The number of user accounts or email addresses associated with the domain.                                                |
| Expiration Date   | The date when the domain or hosting services are set to expire will no longer be accessible.                              |
| Renewal Frequency | The interval or frequency at which the domain or hosting services must be renewed.                                        |
| Suspended         | The temporary deactivation or restriction of the domain or hosting services.                                              |
| Auto Renewal      | A setting that automatically extends the domain or hosting services for another period upon reaching the expiration date. |

![1](/images/portohostingimages/1.png)

2. A popup will appear when you select the  **"Search Filters"** option.

![2](/images/portohostingimages/2.png)

3. The **"Search Filters"** allow you to specify various parameters such as the following:

| Parameter       | Description                                                                                                                                                                                                                                                                                                  |
| :-------------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Domain          | The name and address that uniquely identifies a website on the internet.                                                                                                                                                                                                                                     |
| Status          | The current state or condition of the domain. It can be specified with "In Process", "Active", "Soon to Expire", "Expired-Recoverable", "Expired" , "Disabled", "Cancelled by RSS", "Archived", "Reversed by RSS", "Upgrade in Progress", "Downgrade in Progress", "Adding Accounts" or "Removing Accounts". |
| Hosting Type    | The method or technology used to store and serve website files. It can be specified with "Cloud", "Email", "Productivity Suite", "Professional Cloud", "Shared", "SSL Certificate" or "WebBuilder".                                                                                                          |
| Hosting Brand   | The company or provider that offers the hosting services for the website. It can be specified with "Business", "Comodo" ," Drupal" , " Enterprise", "Google WorkSpace", "Joomla", "Lamp", "Linux", "Magento", "Weebly", "Windows", "Woopress" or "WooCommerce".                                              |
| Hosting Plan    | The specific package or tier of hosting services.                                                                                                                                                                                                                                                            |
| Accounts        | The number of user accounts or email addresses associated with the domain.                                                                                                                                                                                                                                   |
| Expiration Date | The date when the domain or hosting services are set to expire will no longer be accessible.                                                                                                                                                                                                                 |
| Suspended       | The temporary deactivation or restriction of the domain or hosting services. It can be specified with Yes or No.                                                                                                                                                                                             |
| Auto Renewal    | A setting that automatically extends the domain or hosting services for another period upon reaching the expiration date. It can be specified with Yes or No.                                                                                                                                                |
